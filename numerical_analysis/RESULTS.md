# 数值计算结果与解析解对比

## 1. 计算设置

- 自然单位：`hbar = m = 1`
- 无限深势阱：`L = 1`
- 差分网格：100 个内部点，`h = 1 / 101`
- 差分格式：二阶中心差分
- 打靶法：`DOP853` 积分，Brent 法搜索使边界残差为零的能量
- 谐振子：`V(x) = x^2 / 2`，截断区域 `[-5, 5]`

无限深势阱的离散 Hamiltonian 为

```text
H_ii     =  1 / h^2 + V(x_i)
H_i,i+1  = -1 / (2 h^2)
H_i,i-1  = -1 / (2 h^2)
```

解析能级为 `E_n = n^2 pi^2 / 2`。

## 2. 无限深势阱

### 差分法前五个本征值

| n | 差分法 | 解析解 | 相对误差 |
|---:|---:|---:|---:|
| 1 | 4.934404339 | 4.934802201 | 0.008062% |
| 2 | 19.732843640 | 19.739208802 | 0.032246% |
| 3 | 44.381001368 | 44.413219805 | 0.072542% |
| 4 | 78.855032022 | 78.956835209 | 0.128935% |
| 5 | 123.121584205 | 123.370055014 | 0.201403% |

误差随 `n` 增大而上升，因为高能态波长更短，固定网格对波函数曲率的
分辨能力下降。所有前五级误差仍低于 `0.21%`。

### 打靶法前三个本征值

| n | 打靶法 | 解析解 | 绝对误差 |
|---:|---:|---:|---:|
| 1 | 4.934802200552 | 4.934802200545 | 7.27e-12 |
| 2 | 19.739208802208 | 19.739208802179 | 2.93e-11 |
| 3 | 44.413219804971 | 44.413219804902 | 6.84e-11 |

该问题势能为常数、解平滑且积分区间短，因此高精度 ODE 积分配合
Brent 找根可以达到接近机器精度的结果。

## 3. 谐振子基态

| 方法 | 数值结果 | 解析解 | 相对误差 |
|---|---:|---:|---:|
| 差分法 | 0.499693469533 | 0.500000000000 | 0.061306% |
| 打靶法 | 0.500000000078 | 0.500000000000 | 1.55e-8% |

谐振子基态具有偶宇称，打靶时使用 `psi(0)=1`、`psi'(0)=0`，从
`x=0` 积分到 `x=5`。这比从经典禁阻区边界 `x=-5` 起射更稳定。

## 4. 可用于汇报的结论

1. 差分法把微分方程转化为矩阵本征值问题，结构清晰，并且容易推广到
   任意势场；代价是精度受网格分辨率限制。
2. 打靶法直接调节能量以满足边界条件，在本题的一维低能态上精度很高；
   但对高激发态、多维问题和强禁阻区通常更敏感。
3. 两种数值方法都复现了解析结果，说明离散 Hamiltonian 与边界条件实现
   正确。
4. GitHub 的 `hydrogen-orbital` 项目使用氢原子解析波函数并采样
   `|psi|^2` 生成三维电子云。最终展示时可把它作为从一维验证模型到三维
   量子态可视化的扩展，而不应描述成该项目内部使用了本题的差分或打靶法。
